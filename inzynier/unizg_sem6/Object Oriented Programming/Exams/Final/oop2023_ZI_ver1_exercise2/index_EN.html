In the task, it is necessary to implement the program code of the parameterized class Pair and the class SimplePairHashSet.

The class Pair represents a collection of two parameterized elements A and B, in such a way that the first and second elements are defined in the class, but so that the instances of the class Pair are equal regardless of the order of the elements. So Pair with elements ("T",1) would be equal to (1,"T"). Also, Pair can contain null elements and duplicates.

Methods to implement are marked in the UML diagram. Please notice that the Pair class should implement the equals and hashCode methods, keeping in mind that instances are equal regardless of the order of their elements.

SimplePairHashSet represents a set of elements of type Pair. A set by definition may not store duplicates. Your task is to implement the class according to the UML diagram without using the existing interface Set and the existing implementations of the interface Set (HashSet, TreeSet, ...). Instead you should use the existing implementation of the Map interface, which you will define as the private variable internalMap. Class SimplePairHashSet may only have one that one variable. The required methods of the class SimplePairHashSet should have the same functionality as the methods of the class HashSet of the same name (https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/util/HashSet .html).

Attention: All classes and interfaces have visibility set to package private and are in the hr.fer.oop.pair package. When implementing methods in them, you may use additional auxiliary data structures because the speed of program code execution is not important, i.e. the program code does not have to be optimized. Calling any method can't be able to "destroy" the collection (i.e. delete all elements of the collection or similar).

Usage example:

Pair<String, Double> pair1 = new Pair<>("Peter", 2.7);
Pair<Integer, String> pair2 = new Pair<>(7, "Marco");
Pair<Double, String> pair3 = new Pair<>(2.7, "Peter");

System.out.println("pair1 equals pair2: " + pair1.equals(pair2));
System.out.println("pair1 equals pair3: " + pair1.equals(pair3));

System.out.println("pair1 hashcode equals pair2 hashcode: " + (pair1.hashCode() == pair2.hashCode()));
System.out.println("pair1 hashcode equals pair3 hashcode: " + (pair1.hashCode() == pair3.hashCode()));

SimplePairHashSet simplePairHashSet = new SimplePairHashSet();
System.out.println("add pair1: " + simplePairHashSet.add(pair1));
System.out.println("add pair2: " + simplePairHashSet.add(pair2));
System.out.println("add pair3: " + simplePairHashSet.add(pair3));

simplePairHashSet.forEach(pair -> System.out.printf("(%s, %s);", pair.getFirst(), pair.getSecond()));
System.out.println();

System.out.println("simplePairHashSet size: " + simplePairHashSet.size());
System.out.println("simplePairHashSet contains pair1: " + simplePairHashSet.contains(pair1));
System.out.println("simplePairHashSet contains pair3: " + simplePairHashSet.contains(pair3));
System.out.println("simplePairHashSet remove pair3: " + simplePairHashSet.remove(pair3));
System.out.println("simplePairHashSet remove pair3: " + simplePairHashSet.remove(pair3));
System.out.println("simplePairHashSet size: " + simplePairHashSet.size());
